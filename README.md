## 用法
### 1.安装node
不必多说。  

### 2.克隆本仓库
```commandline
git clone https://github.com/LxHTT/MCSLWebsite.git
```
### 3.安装依赖
先导航之本项目根目录。
```commandline
npm install hexo-cli -save
```
### 4.运行
#### 本地运行
```commandline
hexo s
```
#### 构建
```commandline
hexo g
```
如果遇到构建后仍没有变化的情况，可执行以下指令:
```commandline
hexo cl
hexo g
```
构建后的文件在根目录的`public`文件夹中。
