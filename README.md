## 用法
### 1.安装nodejs
https://nodejs.cn/download/

### 2.克隆本仓库
```bash
git clone https://github.com/MCSLTeam/MCSLWebsite.git
```
### 3.安装依赖
先导航到本项目根目录。
```bash
npm i
```
### 4.运行
#### 本地运行
```bash
hexo s
```
#### 构建
```bash
hexo g
```
如果遇到构建后仍没有变化的情况，可执行以下指令:
```bash
hexo cl
hexo g
```
构建后的文件在根目录的`MCSL-Web`文件夹中。
