## 下载MCSL2  

1. 点击[这里](/download)下载最新版本的MCSL2。
2. 下载压缩包解压后，双击 `MCSL2.exe`(Windows平台) 或 `MCSL2.bin`(Linux/Mac平台)启动程序。

::: details Linux/Mac版无法启动？
请打开终端，输入：

```bash
chmod +x MCSL2.bin
```

然后再运行程序。  
实在不行也可下载源码搭建环境运行。
:::
